Guidelines on how to use Sir Enocks Cor CSS Framework.



Margin.

It is .m-*amount* eg .m-10 to set margin to 10px. Available margin amounts are:
  • m-0
  • m-1
  • m-2
  • m-3
  • ...
  • m-15

It is .ml-*amount* eg .ml-10 to set margin-left to 10px. Available margin-left amounts are:

Text Decorations.

Sir Enocks Cor has 4 major text decorations effecs which are text-decoration-style, text-decoration-line, text-decoration-thickness and text-decoration-color. The framework works as follows:

a) No Effects.

(this is normally used in cases where text has some effects by default e.g when using " a tags" where the text is underlined by default.)

It is .deco-f (f=false) to remove all decoration effects from the text.

b) decoration color.

It is .deco-*colorName* eg .deco-blue to set decoration effects to blue color. Available colors are:
  • deco-white
  • deco-red
  • deco-green
  • deco-blue
  • deco-navy

c) decoration style.

It is .deco-*style* eg .deco-wavy to set decoration effects to wavy style. Available styles are:
  • deco-dotted
  • deco-solid
  • deco-double
  • deco-wavy
  • deco-dashed

d) decoration line.

It is .deco-*line* eg .deco-under to underline the text. Available lines are:
  • deco-over-under
  • deco-under
